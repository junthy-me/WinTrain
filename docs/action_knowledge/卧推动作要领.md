# 卧推动作要领

## 杠铃卧推

前置准备：

1. 护腕手套；
2. 充分热身；

步骤：

1. 坐在凳子靠前位置，肩胛骨绷紧后缩，身体倒向凳子；
2. 握杆前三处调整：
   1. 杠铃在脸部的投影位置应该在鼻子和嘴唇之间；
   2. 双手触碰两边架子进行调整确保身体处于正中间的位置；
   3. 以双脚-臀部-两边肩胛骨为五点牢牢地贴紧地面或板凳以构成稳固的五点支撑，胸部挺起，背部自然反弓；
   4. 整体效果是上半身绷紧反弓，双脚踏实地面并往前蹬，屁股贴实凳面，两侧肩胛骨贴实凳面；
3. 握杆姿势调整：
   1. 调整握距使得手臂与杆垂直；
   2. 双手五指抓握住杆，以虎口为轴心做一个小臂内旋，使得杆放在手掌偏下连接手腕的肌肉上（大拇指下方肌肉），杆尽可能沿着虎口向下的方向压在小鱼肌上；此时无名指部分与杆有空隙；
   3. 保持小臂与杆的垂直；
   4. **肩往外旋，使得肩胛骨重新回到下沉后缩的状态（通过虎口和小鱼肌卡住杆，使其在肩外旋时能够继续保持之前的握杆姿势）；**
4. 出杆：
   1. 肩胛骨后缩，深吸一口气（瓦式呼吸，稳定核心）；
   2. 手臂伸直（此时杆脱离架子被垂直举起）；
   3. 肩胛骨下沉后缩（此时肩胛骨最稳定，重点！），带动杠铃往前移动（背部力量起作用），移动到杆和肩胛骨保持在一条水平竖线的位置，手臂垂直于杆，该位置作为卧推动作的起始位置；
   4. 吐气不动；
5. 推举：
   1. 重新深吸一口气（瓦式呼吸）；
   2. 用背部力量将杠铃缓慢往下拉，一直到杠铃触碰到身体为止（全程肩胛骨应该处于后缩且下沉绷着的状态，十分稳定）；
   3. 按照下拉的路径向上推进行还原，还原的过程中为了保证肩胛骨的稳定，需要用力将背部往凳子上顶；（上推还原时双向用力：1. 推的力；2.背部往下顶带来的反作用力）
   4. 推举全程手臂应该与杆始终保持垂直，而杆压在小鱼肌上将重量通过手腕传递到手臂上，而不是手腕承重；
   5. 推举额外补充的注意点：
      1. 推举的过程中，对于肩胛骨而言，胸部会带着其往前，但此时需要后背收缩将肩胛骨尽可能往后拉做对抗，这里是利用背阔肌收缩将肩胛骨往下沉，现象就是推的过程手臂是有个往下摁的一个动作，而不是直推。这里往下摁其实就是背阔肌收缩下沉带来的。
      2. 大臂和腋下的夹角最好保持 45度，最多不要超过 60度，否则肩颊骨肩峰位置会和大臂股骨头产生磨擦，造成“肩峰撞击”。
6. 收杆：
   1. 将杆放到杠铃架上，起身休息；
