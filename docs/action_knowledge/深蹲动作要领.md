# 深蹲动作要领

## 杠铃深蹲

前置准备：

1. 举重鞋或者平板鞋，力量腰带；
2. 充分热身；

步骤：

1. 杠铃位置调整：先将杠铃放到合适高度——应该在锁骨以下，乳头以上的位置，如果没法灵活调节，就低不就高；
2. 起步站姿调整：
   1. 脚往前迈，使得杠铃垂直线落在脚掌中心位置；
   2. 调整双脚之间站距至合适位置（两边膝盖分别指向第二个脚趾的方向，此时最安全稳定）；
   3. 双手对称握住杠铃滚花，双手间距可依据自己的关节灵活度调整，一开始可以握的宽一些（比肩宽）；
3. 深蹲起步出杆：
   1. 肩胛骨后缩，身子略微下蹲往前探，将杠铃放在自己的肩膀后侧肌肉上；
   2. 小手手臂通过“掰杆”动作（想象将杠铃掰成倒 U 型）向下调整，此时会对肩膀上的杠铃施加向下的力，从而让杠铃更紧的固定在肩膀上；
   3. 深吸一口气憋住（瓦式呼吸，气体吸入腹部，舌尖抵住上颚），膝盖打直（杠铃离开杠铃架），依照 左脚往后半步-右脚往后一步-左脚往后半步 进行**平行**退后；
   4. 微调双脚站距和脚尖方向，头抬起来平行望向前方，吐气；
   5. 退的过程中不要低头；
4. 蹲起：
   1. 深吸一口气憋住（瓦式呼吸，核心鼓胀从而增强核心稳定性）；
   2. 髋关节与膝关节同步缓慢下蹲（缓慢是为了持续与重力做对抗，整个下蹲就像用屁股小心翼翼地去触碰下面的火苗）；
   3. 下蹲同时髋关节做外旋动作，但不必真的腿往外旋转开，只是做一个对抗来保证膝盖稳定不内收；下蹲需要保证背部挺直，直到下蹲到一个不能再挺直背下蹲的深度为止；
   4. 蹲到最低处站起，站起过程中保持髋关节外旋对抗、髋膝同步以及背部挺直，整个蹲起过程的肌肉收缩是先将大腿内侧肌肉拉伸绷紧（蹲下），然后让其弹开恢复（起身）；
   5. 蹲起过程中，保持脚趾扣地，用于建立足弓，增强整体的稳定；
5. 回杆：目光平视前方扛着杠铃往前走，直到听到&感受到杠铃两边都碰到了杠铃架即可卸下杠铃；

如何判断深蹲做的好不好？ 下蹲和站起的过程中，杠铃端点是否尽可能地沿着垂直的方向上下，且杠铃端点在地面上的投影点是否在脚底板的中心处。
