## 杠铃划船

### 起始姿势（与罗马尼亚硬拉大致一样）

1. 脚的站距是能够让自己跳的最高的站距
2. 先充分曲髋到合适角度（大腿后侧紧绷度到90%～95%左右，不要 100%），小腿保持和地面垂直
3. 保持背部竖直（不要拱背），手臂放松自然下垂，不用提前掰杆让杆贴近我的身体（与硬拉的最大不同）
4. 后续做动作时，保持身体整个姿势不动；

### 动作要领

1. 起始姿势准备好后，肩胛骨下沉后缩，胳膊自然放松；
2. 吸气（瓦式）憋住，背阔肌发力通过胳膊带动杠铃沿着大腿方向往后拉，拉到杠铃到达肚子位置，然后再将杠铃推回去，吐气；

### 注意点

1. 后侧链张力没有建立起来（起始姿势没有做好），导致在做的过程中，会不自觉地通过伸髋和曲髋来借力；
2. 做的过程中胳膊肘应该是往后划，应该尝试往屁股方向去碰；
3. 往回推的时候，胳膊伸直动作就算完成，不要再往前伸，因为此时肩胛骨会动；
4. 做的过程中不要仰头，保持脊柱中立，吸气的时候收住下巴，让头部和整个背部保持一条直线；

### 杠铃划船变式

| 变式                     | 改进点                                                       | 优缺点                                                       |
| ------------------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| 背角更竖直的划船         | 让背更竖直一些，这样阻力臂缩短，下背压力更小，整个背部能够承担的重量更多（主要是上背）； | 优点： 能够减轻下背压力，整个背部能够承担的重量增加； 缺点： 背阔肌收到的刺激变小； 杠铃做工的行程更短； |
| 背角和地面接近水平的划船 | 背几乎水平                                                   | 优点： 做工距离变长 缺点： 后背压力很大（阻力臂很长），所以一般不做离心，推回的时候直接做自由落体； |
| 通过炮筒来进行辅助划船   | 借助辅助设备，同时能够兼顾背角竖直（对下背压力小）和行程够长两个方面 | 优点： 同时能够兼顾背角竖直和行程够长； 缺点： 功能性要差些，因为是固定的，所以不需要自己去维持重心的平衡； |
