# 高位下拉动作要领

# **坐姿高位下拉**

这个动作主要练背阔肌。你不要想着“用手把杆拉下来”，要想着**用背带着大臂往下走**。

最重要的发力感觉，记住一句话：

**想象腋下塞了两个苹果，下拉的时候用手臂把苹果夹爆。**

什么意思？就是下拉时先别急着用手发力，而是先让肩膀沉下去，再让大臂往身体两侧下压。你一旦有“夹腋下”的感觉，背就更容易发上力。

### 动作要领

坐稳，脚踩实，大腿固定好。

握住杆以后，胸口打开，腰背保持自然，不要弓背。

开始下拉时，先**沉肩**，再让**肘往下走**，把杆拉到上胸附近。

拉到底的时候停一下，感受腋下和背两侧收紧。

放回去要慢，不要直接弹上去。

### 注意点

第一，不要耸肩。肩一耸，力量就容易跑到斜方肌和手臂上。

第二，不要只想着用手拉，要一直想“肘往下压，腋下夹紧”。

第三，不要为了拉更重把身体大幅后仰，轻微后倾就够了。

第四，放回去也要控制，背部要有拉伸感。

### 你训练时可以一直提醒自己

**先沉肩，后下拉；肘往下走，腋下夹紧；不是手在拉，是背在发力。**
